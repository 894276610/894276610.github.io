Markdown基础

Markdown编辑注重键盘操作，省去费时的鼠标操作，享受码字过程以及优质排版。
这篇博客简单记录一下基本的Markdown使用入门

  • 准备工作
  • 基本操作
  • 参考文献
  • 其他花絮

准备工作

VS code有丰富的插件，可以支持Markdown编辑以及预览
其实还有很多专注Markdown文本编辑的软件

  • Typora
  • Joplin
  • Cmd Markdown

如果你选择VS code, 那么可能需要安装以下插件：

  • Markdown Toc 生成目录
  • Markdown All in One 实现基本语法
  • Markdown Preview Enhanced 实现预览效果
    点击VS code 软件内图标输入相应插件并下载

基本操作

  1. 换行
    行末加两个空格再按回车

  2. 加粗，倾斜

    **加粗** => 加粗
    快捷键 Ctrl + B

    *倾斜* => 倾斜
    快捷键 Ctrl + I

  3. 标题格式
    ### 三级标题
    #### 四级标题
    最多支持六级标题

  4. 插入图片
    ![注释](图片地址)
    插入网站图片：
    [![UaC6ns.th.jpg](https://s1.ax1x.com/2020/07/14/UaC6ns.th.jpg)](https://imgchr.com/i/UaC6ns)

  5. 插入公式
    支持Latax
    $$ E=mc^2 $$
    在网站显示公式还需要额外的配置，留个坑

  6. 文字引用
    实例： >引用一段文字

    引用一段文字

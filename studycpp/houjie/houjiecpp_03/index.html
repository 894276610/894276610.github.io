静态 static

静态函数没有this指针
静态函数只能处理静态数据

class Account{
public:
    static double m_rate;  //declaration
    static void set_rate(const double& x){
        m_rate = x;
    }
};

//definition 
//这里要不要给初值都可以  
double Account :: m_rate = 8.0;


int main(){
    //静态函数通过类调用
    Account::set_rate(5.0);

    //静态函数通过产生的实例对象调用
    Account a;
    a.set_rate(7.0);
}

Inheritance with virtual function

在任何一个成员函数之前加
virtual 关键字

  1. non-virtual 不希望子类重新定义 覆写
    没有virtual 关键字
  2. virtual 希望子类重新定义 覆写 virtual void error(const std::string& msg);
  3. pure virtual 子类一定要重新定义
    virtual void draw() const = 0;

自定义话题消息

  1. 定义msg文件 在learning_topic 中 msg文件夹下
    创建Phone.msg 文件

    是 uint 不是 unit 别敲错了

    ## uint8 实在std_msgs 中做定义的第二步需配置 ##
   
string brand
uint8 price  
uint8 sizes  

uint8 big = 0
uint8 small = 1

  2. 在package.xml 中添加功能包依赖
    动态生成程序的依赖
    （这个部分写过就不用重复写辽）

    <build_depend>message_generation</build_depend>  
<exec_depend>message_runtime</exec_depend>

    粘贴到与其相同格式的地方

  3. 在CMakeList.txt添加编译选项
    注意：先把相似的add完再generate

    find_package(... message_generation)  

## Declare ROS Messages, services and actions ##

add_message_files(FILES Person.msg)  
add_message_files(FILES Phone.msg)
generate_messages(DEPENDENCIES std_msgs)  
catkin_package(CATKIN_DEPENDS ... message_runtime)

  4. 在根目录下编译

    catkin_make

  5. （查看生成的代码文件）
    devel/include/learning_topic/Phone.h

创建发布者代码

发布者

#include <ros/ros.h>
#include "learning_topic/Phone.h"

int main(int argc, char** argv)

{

   //initialization

   ros::init(argc, argv, "publish_phone");

   //create NodeHandle

   ros::NodeHandle n;

   //Create a publisher

   ros::Publisher phone_info_pub = n.advertise<learning_topic::Phone>("/phone_info",10);

   //Set the frequency

   ros::Rate loop_rate(1);

   while(ros::ok())

   {

       //Initialize Phone message

       learning_topic::Phone phone_msg;

       phone_msg.brand = "HuaWei";

       phone_msg.price = 2.9;

       phone_msg.sizes  = learning_topic::Phone::big;

       //publish

       phone_info_pub.publish(phone_msg);

       ROS_INFO("Publish Person Info: brand %s price %d size %d",

                          phone_msg.brand.c_str(),phone_msg.price,phone_msg.sizes;

       //rate

       loop_rate.sleep();

   }

       return 0;

}

订阅者代码

#include <ros/ros.h>

#include "learning_topic/Phone.h"



//callback function

void phoneInfoCallback(const learning_topic::Phone::ConstPtr& msg)

{

    // show the message

    // function usage: const char *c_str();

    ROS_INFO("Phone Brand: %s  Price: %d Size: %d",

             msg->brand.c_str(),msg->price,msg->sizes);



}



int main(int argc, char** argv)

{

    //Initialization

    ros::init(argc, argv, "phone_subscriber");



    //create handle

    ros::NodeHandle n;



    //create subscriber

    ros::Subscriber phone_info_sub = n.subscribe("/phone_info", 10,phoneInfoCallback );



    //wait for the message

    ros::spin();



    return 0;

}

配置代码编译规则

learning_topic 的CMakeList ##build##下方

  • 设置需要编译的代码和生成可执行文件
  • 设置链接库
  • 添加依赖项
add_executable(phone_publisher src/phone_publisher.cpp)
target_link_libraries(phone_publisher ${catkin_LIBRARIES}) 
add_dependencies(phone_publisher ${PROJECT_NAME}_generate_messages_cpp)

add_executable(phone_subscriber src/phone_subscriber.cpp)  
target_link_libraries(phone_subscriber ${catkin_LIBRARIES})  
add_dependencies(phone_subscriber ${PROJECT_NAME}_generate_messages_cpp)
